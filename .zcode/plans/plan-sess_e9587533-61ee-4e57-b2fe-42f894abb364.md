# 剩余工作执行计划

## 一、软中断增强（文件回滚 + 多级粒度）

### 目标

当前 soft interrupt 只停了工具、注入 steer，但**已写的文件不还原**。增强后：
- soft interrupt 时自动回滚被中断工具已修改的文件（比 pi 强——pi 也不回滚）
- `interrupt(level)` 三档粒度：`"soft"`（中断工具+回滚文件）| `"hard"`（回滚文件+不继续）| `"abort"`（完全停止）

### 改动点

**文件**: `src/file_snapshot/mod.rs` (FileSnapshotExtension)

在 `on_tool_execution_end` 中加 interrupt 检查：
```rust
if ctx.is_interrupted {
    // 不保存 after snapshot
    // 从 before_states 取出 BeforeState
    match before_state {
        BeforeState::FileCapture { hash } => {
            // 从 ObjectStore 读旧内容写回
            restore_single_file(&store, path, hash).await
        }
        BeforeState::DirCapture => {
            // tree restore 回 turn_start 前的状态
            restore_to_tree(&store, turn_tree_hash).await
        }
        _ => {}
    }
    return Ok(());
}
```

**文件**: `src/agent/extension.rs`

`ToolExecutionContext` 加 `is_interrupted: bool` 字段（现在已有，默认 false）。

**文件**: `src/agent/agent_loop.rs`

`Agent::interrupt()` 加 `level` 参数：
```rust
pub fn interrupt(&self, level: InterruptLevel) {
    self.interrupted.store(true, SeqCst);
    self.interrupt_level.store(level as u8, SeqCst);
}
```

`InterruptLevel` 枚举：
```rust
pub enum InterruptLevel {
    Soft = 0,   // 中断工具 + 回滚文件 + 继续 agent
    Hard = 1,   // 中断工具 + 回滚文件 + 不继续（相当于 abort + restore）
    Abort = 2,  // 直接停止（等同于现有 stop()）
}
```

### 改动量
- `extension.rs`: +1 字段
- `file_snapshot/mod.rs`: +30 行（中断检测 + restore）
- `agent_loop.rs`: +15 行（InterruptLevel + 传 is_interrupted）
- **合计: ~50 行**
