# src/mcp/mod.rs 文件总结

根据前20行代码，这个文件是 **MCP (Model Context Protocol) 客户端模块**，主要功能包括：

1. **连接 MCP 服务器** - 连接用户配置的 MCP server
2. **工具发现** - 发现服务器提供的工具
3. **工具暴露** - 通过 `mcp__<server>__<tool>` 命名规范将工具暴露给 LLM
4. **运行时状态管理** - 维护单个 MCP server 的运行时状态（`ServerEntry`），包括服务器状态和发现的工具列表
5. **类型定义** - 定义了 `McpClient` 类型别名，用于支持并发的 `call_tool` 调用

这是一个中间件层，负责桥接用户、LLM 和外部的 MCP 工具服务器。
